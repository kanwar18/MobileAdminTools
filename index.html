    MobileAdminTools

    Mobile Admin Tools is a RubyMotion project for Salesforce Administrators.

    Specifically, Mobile Admin Tools currently gives Administrators on the go access to their org's users and their users information. Additionally, Administrators can reset users' passwords, set permissions and verify Roles, Profiles, Managers etc.
